# FireBaseを使用したReactメモアプリ

- Authentication
- FireStore DB
- FireBase Hosting

## 機能
- メールアドレス・PWでのログイン・会員登録
- ログアウト
- 退会機能(yet)
- メモ機能(新規作成・編集・削除)

## 次期機能
- WebShare
- タグ分類

## 使用ライブラリ
- yarn
- react-router-dom v6
- styled-components
- MUI Material UI 
- dayjs
